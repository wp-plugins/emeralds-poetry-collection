== Installation ==

1. Upload `emeraldpoem.php` and `emeraldpoem.inc` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the widget "Emeralds diktsamling" somewhere in your blog.
